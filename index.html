Welcome to Radar

An Operation Services Application

Easy updates

Keep customers up to date on what phase their required tools are in from the Ideation phase, right though to Production so as not to cause any nasty surprises!

Drag & Drop Interface

Made an update to a tool? Simply drag it to the relevant circle and it's live for others to see instantly. Need to see more information? Just double tap the circle and get more info in a stylish modal.

XML Data Storage

Circle coordinates are updated straight away and stored in an XML file on the server that the application is running on so no need to worry about hooking up your own DB.
